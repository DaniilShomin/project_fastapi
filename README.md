## How to install  
* Creat database [PostgreSQL](https://github.com/Hexlet/ru-instructions/blob/main/postgresql.md) or sqlite3  
* Create .env according to .env_example  
* Run the installation commands  
```
make install
make build
```
## How to work  
```
make start-server
```
